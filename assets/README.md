## 🖼️ Image Tips
- Certificate images: any size works — they display fully without cropping
- Project images: 800×450px recommended (16:9)
- Formats: .jpg, .jpeg, .png, .webp all work
- If an image is missing, a styled placeholder shows automatically — no errors!

## ➕ Adding New Certificates
Just drop the image file into the right folder, then open certifications.html
and follow the copy-paste instructions in the comments to add a new card.
